# Equipo 03 - Procesos de Innovación para la Ingeniería
## Carrera de Ingeniería Industrial
Universidad Peruana Cayetano Heredia


🌍 Descripción del Equipo
Somos el Equipo 03 del curso de Procesos de Innovación para la Ingeniería 2026-1, conformado por estudiantes de la carrera de Ingeniería Industrial.

Nuestro objetivo es convertir ideas extraordinarias en una realidad que genere valor en nuestra sociedad, bajo un proceso sistemático, estructurado y gestionable.
